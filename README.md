## How to download the project:

```bash
# Clone the repository
$ git clone https://github.com/rodrigoaraujo7/tanalista.git

# Enter directory
$ cd tanalista
```

## Usage

1. Run `npm install`.<br />
2. Run `npm run dev` and access `http://localhost:5173/`.<br />
